# TOP SOUND — сайт установочной студии

Современный адаптивный одностраничный сайт для установочной студии автозвука **TOP SOUND** (Пенза).
Чистый HTML + CSS + JavaScript, без сборки и зависимостей. Готов к публикации на **Vercel** или **Netlify**.

## Возможности

- Главный экран с заголовком, подзаголовком и кнопками «Записаться» / «Позвонить»
- Блоки: О компании, Преимущества, Услуги и цены, Онлайн-запись, Адрес, Связь
- Форма онлайн-записи: имя, телефон (с маской), email, **марка** и **модель** авто
  (выпадающие списки + ручной ввод), услуга, комментарий
- После отправки — сообщение «Спасибо! Мы свяжемся с вами в ближайшее время» (заглушка, без реальной отправки)
- Две встроенные Яндекс.Карты с метками студий
- Плавные анимации появления блоков при прокрутке, hover-эффекты
- Адаптив: одинаково хорошо на ПК и смартфонах
- SEO: мета-теги, Open Graph, Schema.org (AutoRepair), robots.txt, sitemap.xml

## Структура

```
index.html      — разметка
styles.css      — стили (красно-чёрно-белая тема)
script.js       — логика формы, марки/модели, анимации, меню
vercel.json     — конфиг для Vercel
netlify.toml    — конфиг для Netlify
robots.txt
sitemap.xml
```

## Локальный просмотр

Просто откройте `index.html` в браузере. Либо запустите простой сервер:

```bash
npx serve .
# или
python3 -m http.server 8000
```

## Публикация

### Vercel
1. Загрузите содержимое папки в репозиторий GitHub.
2. На vercel.com → **Add New Project** → импортируйте репозиторий.
3. Framework Preset: **Other**, Build Command — пусто, Output Directory — `.`
4. Deploy.

### Netlify
1. Загрузите в GitHub.
2. На netlify.com → **Add new site → Import an existing project**.
3. Publish directory: `.`  → Deploy.

Либо перетащите папку в Netlify Drop: https://app.netlify.com/drop

## Настройка под себя

- **Фото услуг**: в `styles.css` блоки `.service__media--*` используют градиенты-заглушки.
  Замените на реальные фото: `background-image: url('images/install.jpg');`
- **Фон главного экрана (hero)**: добавьте свою фотографию авто/автозвука в `.hero`.
- **Email**: замените `info@topsound.ru` в `index.html` на реальный.
- **Марки/модели авто**: список в начале `script.js` (объект `CAR_DATA`).
- **Реальная отправка заявок**: в `script.js`, обработчик `form.submit` —
  замените `console.log` на отправку (Formspree, Telegram Bot, email-сервис и т.п.).
